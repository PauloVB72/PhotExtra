A code to implement photometry in different data surveys, eliminating all kinds of stellar artifacts. Its main use is for SED fitting codes.

Surveys implemented:

* PS1
* SDSS
* GALEX
* 2MASS
* WISE
* unWISE
* LegacySurvey
* VISTA
* SPLUS
* UKIDSS
